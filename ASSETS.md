# Asset Notes

This repository mixes source code, generated runtime data, and bundled art/media assets.

## Source Code

The source code is released under the ISC license in [LICENSE](./LICENSE).

## Bundled Raster Assets

The following bundled raster assets are required for the current game presentation:

- `public/assets/backgrounds/ash-plains-base-tile.png`
- `public/assets/backgrounds/ash-plains-overlay-tile.png`
- `public/assets/backgrounds/ember-hollow-base-tile.png`
- `public/assets/backgrounds/ember-hollow-overlay-tile.png`
- `public/assets/backgrounds/moon-crypt-base-tile.png`
- `public/assets/backgrounds/moon-crypt-overlay-tile.png`
- `public/assets/vfx/projectiles-atlas.png`
- `public/assets/vfx/effects-atlas.png`
- `docs/assets/gameplay_demo.webp`

These files are distributed with the repository so the project can run and so the README demo remains representative. Do not assume they are covered by the ISC code license unless a file or commit explicitly says so.

If you plan to reuse, redistribute, or commercialize bundled art/media outside this repository, confirm ownership and provenance first and replace anything you cannot verify.

## Procedural And Runtime-Generated Assets

Procedural visuals and audio generated at runtime from the checked-in source code follow the source-code license because they are derived from the code path itself rather than from separately bundled media files.
